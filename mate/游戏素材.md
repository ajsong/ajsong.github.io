# 游戏素材汇总

# 一、通用大型免费素材库

## 1. Kenney
- **链接**: <https://kenney.nl/assets>
- **许可**: CC0（完全免费可商用）
- **特点**: UI / 2D / 3D / 音效全覆盖；风格统一；Godot 官方推荐
- **适合**: 直接做完整游戏，UI、地形、图标、按钮
- **来源**: 素材1、素材2

## 2. OpenGameArt
- **链接**: <https://opengameart.org>
- **特点**: 最大开源游戏资源社区；RPG / 地牢 / UI / 音效应有尽有
- **注意**: 风格不统一，但数量巨大
- **适合**: 找"任何你想要的东西"
- **来源**: 素材2

## 3. itch.io 免费素材区
- **链接**: <https://itch.io/game-assets/free>
- **特点**: indie 作者上传，风格现代甚至商业级，大量完整 RPG Pack
- **适合**: 找整套游戏资源包
- **来源**: 素材1、素材2

## 4. Quaternius
- **链接**: <https://quaternius.com>
- **特点**: 全部免费 CC0；建筑 / 树木 / 角色 / 3D 模型风格统一
- **适合**: 快速搭建完整世界
- **来源**: 素材1、素材2

## 5. KayKit
- **链接**: <https://kaylousberg.itch.io/>
- **特点**: 专门做"可拼接 RPG 世界"；地牢 / 城镇 / 角色；风格统一
- **适合**: 幻想 RPG 项目，尤其等距视角
- **来源**: 素材1、素材2

## 6. CraftPix（免费区）
- **链接**: <https://craftpix.net/freebies/>
- **特点**: UI + RPG + 2D 地图丰富；fantasy 风格强；有大量等距地图素材
- **适合**: 等距 RPG 项目
- **来源**: 素材2

## 7. PolyPizza
- **链接**: <https://poly.pizza>
- **特点**: CC0 3D 模型
- **适合**: 快速获取 3D 资产
- **来源**: 素材2

# 二、UI / 图标 / HUD 专用

## 8. Game-Icons.net
- **链接**: <https://game-icons.net>
- **特点**: 4000+ 图标，SVG 格式可无限放大；RPG 技能图标标准来源
- **适合**: 技能 / buff / debuff / UI 按钮
- **来源**: 素材2

## 9. FreeGameUI
- **链接**: <https://freegameui.net>
- **特点**: CC0 UI pack；SVG 可编辑；按钮 / 面板 / 进度条 / 图标齐全
- **适合**: 直接做完整 HUD
- **来源**: 素材2

## 10. Game UI Database
- **链接**: <https://www.gameuidatabase.com>
- **特点**: 不是下载站，而是设计参考；收集商业游戏 UI
- **适合**: 做"像 Diablo / Hades 的 UI"时参考
- **来源**: 素材2

---

# 三、角色 / 怪物 / 动画

## 11. Mixamo
- **链接**: <https://www.mixamo.com>
- **特点**: 免费 3D 角色动画；自动绑定骨骼；可直接导出 Godot / Blender
- **适合**: 人物动作（攻击 / 跑步 / 死亡）
- **来源**: 素材2

## 12. Pixel Frog（itch.io）
- **链接**: <https://pixelfrog-assets.itch.io/>
- **特点**: 免费 2D RPG 套件；动画完整；商业级质量
- **适合**: 角色 / 怪物 / 动画精灵
- **来源**: 素材2

---

# 四、场景 / 地图 / 地形

## 13. Poly Haven
- **链接**: <https://polyhaven.com>
- **特点**: CC0 HDRI / 纹理 / 模型；超高清
- **适合**: 地面材质 / 石头 / 光照环境
- **来源**: 素材2

## 14. AmbientCG
- **链接**: <https://ambientcg.com>
- **特点**: 8K PBR 贴图；木头 / 石头 / 草地 / 金属
- **适合**: Godot 4 PBR 场景
- **来源**: 素材2

## 15. Texture Haven
- **链接**: <https://texturehaven.com>
- **特点**: 同 AmbientCG 体系，专注高质量贴图
- **适合**: 无缝纹理
- **来源**: 素材2

## 16. GameDev Market（免费区）
- **链接**: <https://www.gamedevmarket.net>
- **特点**: UI / RPG / fantasy pack
- **适合**: 场景装饰与氛围素材
- **来源**: 素材2

---

# 五、音效 / BGM

## 17. Freesound
- **链接**: <https://freesound.org>
- **特点**: 最大音效库；UI / 魔法 / 怪物音效齐全
- **适合**: 日常音效素材
- **来源**: 素材2

## 18. MixKit
- **链接**: <https://mixkit.co>
- **特点**: 免费游戏音效；UI click / sword / magic
- **适合**: 2D 音效
- **来源**: 素材2

## 19. FreePD
- **链接**: <https://freepd.com>
- **特点**: 全部 CC0；RPG / dungeon / boss music
- **适合**: 背景音乐
- **来源**: 素材2

## 20. Incompetech
- **链接**: <https://incompetech.com>
- **特点**: Kevin MacLeod 作品；大量经典游戏 BGM
- **适合**: 背景音乐
- **来源**: 素材2
