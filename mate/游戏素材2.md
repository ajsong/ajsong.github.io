# 一、通用大型免费素材库（必收藏）

## 1. Kenney（最推荐）

[https://kenney.nl/assets](https://kenney.nl/assets)

特点：

* CC0（完全免费可商用）
* UI / 2D / 3D / 音效全都有
* 风格统一（非常重要）
* Godot 官方也推荐

适合：

> 直接做完整游戏

---

## 2. OpenGameArt

[https://opengameart.org](https://opengameart.org)

特点：

* 最大开源游戏资源社区
* RPG / 地牢 / UI / 音效都有
* 风格不统一（但量巨大）

适合：

> 找“任何你想要的东西”

---

## 3. itch.io 免费素材区

[https://itch.io/game-assets/free](https://itch.io/game-assets/free)

特点：

* indie 作者上传
* 风格非常现代（甚至商业级）
* 很多完整 RPG pack

适合：

> 找“整套游戏资源包”

---

## 4. CraftPix（免费区）

[https://craftpix.net/freebies/](https://craftpix.net/freebies/)

特点：

* UI + RPG + 2D 地图很多
* fantasy 风格很强
* 有很多“等距地图素材”

适合：

> 你这种等距 RPG 项目

---

# 🎮 二、UI / 图标 / HUD 专用

## 5. Game-Icons.net

[https://game-icons.net](https://game-icons.net)

特点：

* 4000+ 图标
* SVG格式（可无限放大）
* RPG技能图标标准来源

适合：

* 技能
* buff
* debuff
* UI按钮

---

## 6. FreeGameUI（非常推荐）

[https://freegameui.net](https://freegameui.net)

特点：

* CC0 UI pack
* SVG 可编辑
* 直接可以做完整HUD

包括：

* 按钮
* 面板
* 进度条
* 图标

---

## 7. Game UI Database（设计参考）

[https://www.gameuidatabase.com](https://www.gameuidatabase.com)

特点：

* 不是下载，是参考
* 收集商业游戏 UI

适合：

> 做“像 Diablo / Hades 的 UI”
